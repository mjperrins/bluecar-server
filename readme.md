# bluecar-server

```
npm install
npm install apiconnect -g
npm install bluemix-generator -g
```

## Prepare Client
```
cd client
bower install
```

## Edit API
```
apic edit 
```

## Update new JSON Data and Images
```
bluegen
```

## Run Server
```
node server/server
```

## Server
Access `http://localhost:3000'
